# probot-irc

> A GitHub App built with [Probot](https://github.com/probot/probot) that creates a _very_ humble (super beta right now) irc bot that announces repo activity in a channel:

<img src='example.png' />

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```
